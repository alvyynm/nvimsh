# nvimsh
My trusty Neovim setup

## Key mappings

`Space + e` → Toggle file explorer

Space + ff → Find files

Space + fg → Live grep (search text in files)

Space + fb → Find buffers

Space + fh → Help search
gd → Go to definition (LSP)
gr → list references.

gi → go to implementation.

K → Hover docs (LSP)

Space + rn → Rename symbol
